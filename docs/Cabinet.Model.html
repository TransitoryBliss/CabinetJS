Cabinet. Model

The Model instance. A model instance is created from the static model. *Do NOT use the new keyword*.
Source:
  • model.js, line 5

Example

var User = Cabinet.createModel({ 
	username: Cabinet.datatype.STRING 
});
var user = User.create({ 
	username: "myUsername" 
}); 
user.set("username", "anotherUsername");
console.log(user instanceof Cabinet.Model) // => true
